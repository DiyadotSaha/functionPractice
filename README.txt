DESCRIPTION

This program contains multiple functions that perform various activities. 
- func 1:
    counts the number of 0s, positive numbers and negative numbers. 
- func 2
    rotates a given array by one. 
- func 3
    counts the number of non-alphabetic number in a given string excluding spaces. 
- func 4
    deletes 0s from a given string.
- func 5
    reverses the cases of a given C-string
